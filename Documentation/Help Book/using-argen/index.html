ARGen Help


The project window shows the main configuration options on the left. Generate Project will validate the tables and generate the project! More details about each section are broken down by page in the documentation.

When the project window opens there will be a short delay as ARGen scans the database schema and validates the tables. Any errors encountered will be presented to the developer. Before a project can be generated these errors must be corrected or the tables with the errors can be turned off on the appropriate setting panel.
