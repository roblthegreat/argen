ARGen Help


The Project panel has the main configuration options for the project. These options are self explanatory and required to generate a project.

From the project panel the developer can change the project type and project API setting.

Console projects do not generate interface controls. This is indicated by the âš  symbol.
iOS projects do not generate interface controls and do not include the Audit Trail option.

The VerifyClass option will add the new VerifyClass feature to the generated Register method. VerifyClass will check that the properties on the table class exist as columns in the database during the Register function.
Include AuditTrail will add the AuditTrail module and make some configuration changes to ActiveRecord.
Include DatabaseUpdates will add the DBUpdates module to the generated project, and call it during the OpenDB method.
Use GMTOffset of zero with Date objects will correct Date objects GMTOffset and set a value of 0 for saving to the database.
Use SQLDatabaseMBS adapter is available for database connections that have an adapter implemented to use SQLDatabaseMBS instead of Xojo classes.
