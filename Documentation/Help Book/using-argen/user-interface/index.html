ARGen Help


The User Interface panel allows a developer to configure the generated user interface options. Like the class setup view, the checkbox next to each table will enable or disable processing for that table.

Control Column: Select which type of Xojo control is generated. The appropriate kind (TextField vs. WebTextField) will be used when the project is generated.
List Column: When the checkbox is selected, this property will appear in the Listbox that is generated for the table list view. A disabled column will not appear in the list view.
Label Text: This is the text placed on the label next to the control. It cannot be empty.

Prepare label text for localization
This option will place the "Label Text" value as a localized string in the generated Strings module instead of the actual label text value. This can be useful for preparing a project to be localized.

Table Options
Click this button to select the field that the list view will be sorted by initially. Click it again, or select another table, to return to the field list view.

Demo Mode
Demo mode users can download a sample project generated from the Chinook SQLite database from the Strawberry Software website to evaluate the user interface generation features. Download Sample Project
