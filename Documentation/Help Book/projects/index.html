ARGen Help


The Projects List window shows the saved projects in ARGen. Projects are connected to a database. The database type is indicated by the circle identity icon next to the project name. SQLite connections show the last known file path or "localhost" for privacy if the database is encrypted. Server connections show the host property.

To create a new project, click the New button under the Project heading. Select a project and click Edit to continue working on an existing project. Similarly, select a project and click Delete to delete the project.
